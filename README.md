# TOURNEPAL  
**JavaFX-Based Desktop Nepal Tourism Management System with Cultural & Festival Integration**

TOURNEPAL is a desktop tourism application built with **pure JavaFX (no FXML)** focused on promoting tourism in **Nepal**.  
The app highlights **cultural significance**, **festivals**, and **special discounts** tied to Nepali festive seasons.  
It also supports **multi-language UI** using a **key–value based translation system**.

---

## Key Features

- **Desktop Tourism Management System**
  - View and manage tour packages around popular destinations in Nepal  
  - Package details: destination, duration, price, highlights, activities, etc.  
  - Simple booking flow for users

- **Cultural Integration**
  - Information about major **Nepali festivals** (e.g., Dashain, Tihar, Holi, Teej, etc.)  
  - Tourism packages themed around cultural events and regions  
  - Descriptions of local culture, traditions, and heritage sites

- **Festive Discounts**
  - Special **festival-based offers & seasonal discounts**  
  - Discount labels on tour packages during specific festival periods  
  - Logic to apply or show discounts depending on time/festival

- **Multi-Language Support (Key–Value Translation)**
  - Language strings handled via **key–value pairs** (e.g., `home.title=Home`, `menu.file=File`)  
  - Easy to switch between languages (for example, **English** and **Nepali**)  
  - Centralized translation management, making it simple to add new languages

- **No FXML – Pure Java Code UI**
  - Entire UI is created and managed programmatically in **Java**  
  - Windows, scenes, layouts, and controls are built through JavaFX APIs  
  - Simplifies deployment (no separate FXML files)

---

## Tech Stack

- **Language:** Java  
- **GUI Framework:** JavaFX  
- **Architecture:** Desktop application (no web)  
- **UI Definition:** Pure Java (no FXML files)  
- **Localization:** Key–value based translation system
